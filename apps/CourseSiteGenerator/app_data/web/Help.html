Course Site Generator is a desktop Java application for creating and editing Course Web Sites. The app presents 5 tabs two tables, Each one of them represents a sections from the course site. Site Tab allows entering category, basic information and styles of the site. Syllabus Tab allows entering any information relates syllabus. Meeting Times Tab allows entering information about recitations.one with a list of the Teaching Assistants (TAs), to which one can add new names, and a list of the scheduled office hours for those TAs. By selecting a TA and then toggling 1/2 hour time blocks one can set TA office hour reservations. The application saves data to a JSON format that can be used by Web pages for displaying TA office hours.

Step 1: Step 1: Input all course info the tabs

Go over all the tabs and input all the informations of your course site

Step 2: Save the Course site

Once you've completed your work, you can press the save button and give your file a name. Note that the name in the app text field will be the name used in the Recent Work list of the Welcome dialog.

Step 3: Export the Course site

Once you've completed your work, you can press the save button and give your file a name. Note that the name in the app text field will be the name used in the Recent Work list of the Welcome dialog.You can export the course site to obtain files that can be use to display a website base on your course site data.
